Objectives

docAider's Objectives.

The goal of docAider to automate the management of open source repositories, including maintaining and updating GitHub documentation. This page outlines the essential requirements crucial for the development, implementation, evaluation, and long-term sustainability of the docAider.

Requirements

  • Understand LLM/SLM, Microsoft Semantic Kernel, Microsoft Autogen and Windows AI Studio: The system should inetegrate these tools when developing the solution.
  • Automate Documentation Maintenance - The system should automatically and accurately check and update GitHub documentation to ensure it is current and accurate.
  • Open Source Repository Management: The final solution should help automate the overall management of open source repositories
